Module 6: Combining Loops and Conditionals

Learn how to combine loops with if, elif, and else statements to make decisions while repeating actions!

What Are Loops and Conditionals?

Loops allow us to repeat actions multiple times, and conditionals (like if, elif, and else) let us make decisions based on certain conditions.

In this lesson, we’ll combine them to do something fun like making decisions during a journey, or checking different things during a loop.

Example: A Jungle Adventure

Imagine Python is walking through a jungle, and it needs to decide what to do at each step:

steps = 0
while steps < 5:
    print("Step " + str(steps + 1))
    
    if steps == 2:
        print("An obstacle is in the way!")
    elif steps == 4:
        print("The path is clear!")
    else:
        print("Keep walking!")
        
    steps += 1

Here’s what happens in this example:

  • Python walks 5 steps.
  • At step 2, it encounters an obstacle and needs to stop.
  • At step 4, it sees that the path is clear!
  • For every other step, Python just keeps walking.

Let’s Practice!

Write a loop that helps Python walk through the jungle path. If the step is 3, it encounters a snake. If it's step 5, it finds a treasure! Otherwise, Python just keeps walking.

Here’s a starting point:

steps = 0
while steps < 6:
    print("Step " + str(steps + 1))
    
    if steps == 3:
        print("A snake appears! Python needs to be careful.")
    elif steps == 5:
        print("Yay! Python found a treasure!")
    else:
        print("Keep walking!")
        
    steps += 1

Challenge

Write a program where Python walks through a forest path, but every time it reaches a step that is a multiple of 2, it encounters a bird. When it reaches a step that’s a multiple of 3, it finds a river. What does Python do at each step?

steps = 0
while steps < 10:
    print("Step " + str(steps + 1))
    
    if steps % 2 == 0:
        print("A bird flies by!")
    elif steps % 3 == 0:
        print("A river appears!")
    else:
        print("Keep walking!")
        
    steps += 1
