Learn how to create reusable blocks of code to save time and make your programs cleaner!

What Are Functions?

Functions are like magic boxes! When you give them something, they give you something back. This helps you avoid repeating the same actions over and over again in your code.

Imagine you have a magic box that always gives you a toy when you ask. You don’t need to say how to make the toy every time, just ask for it and it’s ready!

How to Create a Function

In Python, we use def to create a function. The word def stands for "define," which means we are telling Python to create this magic box!

def say_hello():
    print("Hello, friend!")

To use this function, you just need to call it like this:

say_hello()  # This will print "Hello, friend!"

Using Functions with Input

We can also give our function some information to work with. This is called an "argument." For example, if we want to greet someone by name, we can create a function that takes a name and says "Hello, [name]!"

def greet(name):
    print("Hello, " + name + "!")

Now, we can give it a name and it will greet that person:

greet("Alice")  # This will print "Hello, Alice!"
